# 管理和激活虚拟环境

## 操作步骤

1. 点击左侧栏中的虚拟环境进行激活
2. 点击删除按钮

## 预期结果

能够正常激活并切换到虚拟环境
能够正常删除虚拟环境

## 实际结果

能够正常激活并切换到虚拟环境
能够正常删除虚拟环境
