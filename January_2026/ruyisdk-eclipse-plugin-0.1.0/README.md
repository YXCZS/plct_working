# 针对 RuyiSDK Eclipse插件 的测试

## 测试说明

以手动测试的方法，验证Eclipse插件的核心功能与关键路径。

## 安装方法

- 从仓库页面下载最新构建的压缩包，解压获得 `ruyisdk-eclipse-plugins-0.1.0.zip`后在Eclipse中进行安装

## 环境配置

+ OS: Ubuntu 25.04 (Plucky Puffin)
+ cpu: 16 核
+ 内存：4G
+ 镜像：https://releases.ubuntu.com/25.04/ubuntu-25.04-desktop-amd64.iso
